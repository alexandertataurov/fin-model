# Typography

Our typography system provides consistent, accessible, and hierarchical text styles that create clear information architecture and improve readability across all devices.

## Overview

The typography system is built on semantic naming conventions and accessibility-first principles. Each text style serves a specific purpose in the interface, from headlines to body text.

## Token Definitions

### Font Families

```typescript
tokens.typography.fontFamily = {
  display: ['Inter', 'system-ui', 'sans-serif'], // Headlines, large text
  sans: ['Inter', 'system-ui', 'sans-serif'], // Body text, UI elements
  mono: ['JetBrains Mono', 'monospace'], // Code, technical content
  elegant: ['Playfair Display', 'serif'], // Premium content, quotes
};
```

### Font Sizes

```typescript
tokens.typography.fontSize = {
  xs: '0.75rem', // 12px - Captions, metadata
```

## Usage Rules

### 1. **Font Family Usage**

- **Display**: Headlines, hero text, large titles
- **Sans**: Body text, UI elements, navigation
- **Mono**: Code blocks, technical content, data
- **Elegant**: Premium content, quotes, special text

### 2. **Font Size Hierarchy**

- **4xl**: Main page titles, hero sections
- **3xl**: Section headers, major divisions
- **2xl**: Subsection headers, card titles
- **xl**: Component headers, form labels
- **lg**: Large body text, emphasis
- **base**: Standard body text, default size
- **sm**: Small text, captions, metadata
- **xs**: Tiny text, labels, timestamps

### 3. **Font Weight Usage**

- **Light (300)**: Subtle text, large headlines
- **Normal (400)**: Body text, default weight
- **Medium (500)**: Emphasis, labels, buttons
- **Semibold (600)**: Headings, important text
- **Bold (700)**: Strong emphasis, primary headings

### 4. **Line Height Guidelines**

- **Tight (1.25)**: Headlines, short text
- **Snug (1.375)**: Subheadings, medium text
- **Normal (1.5)**: Standard body text
- **Relaxed (1.625)**: Long-form content
- **Loose (2)**: Spacious layouts, poetry

## Best Practices

### 1. **Hierarchy Consistency**

```tsx
// ✅ Good - Clear hierarchy
<h1 style={{
  fontFamily: tokens.typography.fontFamily.display.join(', '),
  fontSize: tokens.typography.fontSize['3xl'],
  fontWeight: tokens.typography.fontWeight.bold,
  lineHeight: tokens.typography.lineHeight.tight
}}>
  Main Heading
</h1>

<h2 style={{
  fontFamily: tokens.typography.fontFamily.sans.join(', '),
  fontSize: tokens.typography.fontSize.xl,
  fontWeight: tokens.typography.fontWeight.semibold,
  lineHeight: tokens.typography.lineHeight.snug
}}>
  Sub Heading
</h2>
```

### 2. **Readability Optimization**

```tsx
// ✅ Good - Optimized for reading
<p
  style={{
    fontFamily: tokens.typography.fontFamily.sans.join(', '),
    fontSize: tokens.typography.fontSize.base,
    lineHeight: tokens.typography.lineHeight.relaxed,
    maxWidth: '65ch',
    color: tokens.colors.secondary[700],
  }}
>
  Long-form content with optimal line length and spacing.
</p>
```

### 3. **Responsive Typography**

```tsx
// ✅ Good - Responsive sizing
<h1
  style={{
    fontSize: tokens.typography.fontSize['2xl'],
    '@media (min-width: 768px)': {
      fontSize: tokens.typography.fontSize['3xl'],
    },
    '@media (min-width: 1024px)': {
      fontSize: tokens.typography.fontSize['4xl'],
    },
  }}
>
  Responsive Heading
</h1>
```

### 4. **Accessibility Compliance**

```tsx
// ✅ Good - Accessible text
<p
  style={{
    fontSize: tokens.typography.fontSize.base,
    lineHeight: tokens.typography.lineHeight.relaxed,
    color: tokens.colors.secondary[700],
    maxWidth: '65ch', // Optimal line length
  }}
>
  Accessible text with proper contrast and line length.
</p>
```

## How-To Guides

### Creating a Text Style System

```tsx
import { tokens } from '@/design-system/tokens';

const textStyles = {
  headline: {
    fontFamily: tokens.typography.fontFamily.display.join(', '),
    fontSize: tokens.typography.fontSize['3xl'],
    fontWeight: tokens.typography.fontWeight.bold,
    lineHeight: tokens.typography.lineHeight.tight,
    letterSpacing: tokens.typography.letterSpacing.tight,
  },
  subheadline: {
    fontFamily: tokens.typography.fontFamily.sans.join(', '),
    fontSize: tokens.typography.fontSize.xl,
    fontWeight: tokens.typography.fontWeight.semibold,
    lineHeight: tokens.typography.lineHeight.snug,
    letterSpacing: tokens.typography.letterSpacing.normal,
  },
  title: {
    fontFamily: tokens.typography.fontFamily.sans.join(', '),
    fontSize: tokens.typography.fontSize.lg,
    fontWeight: tokens.typography.fontWeight.semibold,
    lineHeight: tokens.typography.lineHeight.snug,
    letterSpacing: tokens.typography.letterSpacing.normal,
  },
  body: {
    fontFamily: tokens.typography.fontFamily.sans.join(', '),
    fontSize: tokens.typography.fontSize.base,
    fontWeight: tokens.typography.fontWeight.normal,
    lineHeight: tokens.typography.lineHeight.relaxed,
    letterSpacing: tokens.typography.letterSpacing.normal,
  },
  caption: {
    fontFamily: tokens.typography.fontFamily.sans.join(', '),
    fontSize: tokens.typography.fontSize.sm,
    fontWeight: tokens.typography.fontWeight.normal,
    lineHeight: tokens.typography.lineHeight.normal,
    letterSpacing: tokens.typography.letterSpacing.wide,
  },
  code: {
    fontFamily: tokens.typography.fontFamily.mono.join(', '),
    fontSize: tokens.typography.fontSize.sm,
    fontWeight: tokens.typography.fontWeight.normal,
    lineHeight: tokens.typography.lineHeight.normal,
    letterSpacing: tokens.typography.letterSpacing.normal,
  },
};

// Usage
const Text = ({ variant = 'body', children, ...props }) => (
  <div style={{ ...textStyles[variant], ...props }}>{children}</div>
);
```

### Creating a Heading Component

```tsx
interface HeadingProps {
  level: 1 | 2 | 3 | 4 | 5 | 6;
  children: React.ReactNode;
  className?: string;
}

const Heading = ({ level, children, className }: HeadingProps) => {
  const getStyles = () => {
    switch (level) {
      case 1:
        return {
          fontFamily: tokens.typography.fontFamily.display.join(', '),
          fontSize: tokens.typography.fontSize['3xl'],
          fontWeight: tokens.typography.fontWeight.bold,
          lineHeight: tokens.typography.lineHeight.tight,
          letterSpacing: tokens.typography.letterSpacing.tight,
        };
      case 2:
        return {
          fontFamily: tokens.typography.fontFamily.sans.join(', '),
          fontSize: tokens.typography.fontSize['2xl'],
          fontWeight: tokens.typography.fontWeight.bold,
          lineHeight: tokens.typography.lineHeight.tight,
          letterSpacing: tokens.typography.letterSpacing.normal,
        };
      case 3:
        return {
          fontFamily: tokens.typography.fontFamily.sans.join(', '),
          fontSize: tokens.typography.fontSize.xl,
          fontWeight: tokens.typography.fontWeight.semibold,
          lineHeight: tokens.typography.lineHeight.snug,
          letterSpacing: tokens.typography.letterSpacing.normal,
        };
      case 4:
        return {
          fontFamily: tokens.typography.fontFamily.sans.join(', '),
          fontSize: tokens.typography.fontSize.lg,
          fontWeight: tokens.typography.fontWeight.semibold,
          lineHeight: tokens.typography.lineHeight.snug,
          letterSpacing: tokens.typography.letterSpacing.normal,
        };
      case 5:
        return {
          fontFamily: tokens.typography.fontFamily.sans.join(', '),
          fontSize: tokens.typography.fontSize.base,
          fontWeight: tokens.typography.fontWeight.medium,
          lineHeight: tokens.typography.lineHeight.normal,
          letterSpacing: tokens.typography.letterSpacing.normal,
        };
      case 6:
        return {
          fontFamily: tokens.typography.fontFamily.sans.join(', '),
          fontSize: tokens.typography.fontSize.sm,
          fontWeight: tokens.typography.fontWeight.medium,
          lineHeight: tokens.typography.lineHeight.normal,
          letterSpacing: tokens.typography.letterSpacing.normal,
        };
    }
  };

  const Tag = `h${level}` as keyof JSX.IntrinsicElements;

  return (
    <Tag style={getStyles()} className={className}>
      {children}
    </Tag>
  );
};
```

```tsx
const FormLabel = ({
```

## Accessibility Guidelines

### 1. **Font Size Accessibility**

- Minimum font size: 16px for body text
- Ensure text remains readable when zoomed to 200%
- Test with different screen resolutions

### 2. **Line Length**

- Optimal line length: 45-75 characters
- Maximum line length: 65 characters for body text
- Use appropriate line height for readability

### 3. **Contrast Ratios**

- Normal text: Minimum 4.5:1 contrast ratio
- Large text: Minimum 3:1 contrast ratio
- Test with color contrast analyzers

### 4. **Font Loading**

- Use font-display: swap for better performance
- Provide fallback fonts
- Consider font loading strategies

### 5. **Screen Reader Support**

- Use semantic HTML elements
- Maintain proper heading hierarchy
- Provide alternative text for decorative fonts

## Testing Checklist

- [ ] All text meets minimum contrast ratios
- [ ] Font sizes are readable on all devices
- [ ] Line lengths are optimal for reading
- [ ] Heading hierarchy is logical
- [ ] Font loading doesn't cause layout shifts
- [ ] Text is accessible to screen readers
