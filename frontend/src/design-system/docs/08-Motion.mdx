# Motion

Our motion system provides smooth, purposeful animations that enhance user experience while maintaining accessibility and performance. Each animation is designed to guide user attention, provide feedback, and create delightful interactions.

## Overview

The motion system consists of:

- **Easing Curves**: Natural, physics-based timing functions
- **Duration Scales**: Consistent timing for different interaction types
- **Animation Patterns**: Predefined animations for common use cases
- **Performance Optimized**: Hardware-accelerated animations
- **Accessibility Compliant**: Respects user motion preferences

## Motion Tokens

## Usage Rules

### 1. **Easing Curve Usage**
- **linear**: No easing, mechanical movements
- **ease**: Default for most transitions
- **easeIn**: Elements appearing, growing
- **easeOut**: Elements disappearing, shrinking
- **easeInOut**: Complex movements, state changes
- **smooth**: Material Design standard, natural feel
- **bounce**: Playful interactions, notifications
- **elastic**: Attention-grabbing, important actions

### 2. **Duration Guidelines**
- **instant (0ms)**: Immediate state changes
- **fast (100ms)**: Quick feedback, hover states
- **normal (200ms)**: Standard transitions, most interactions
- **slow (300ms)**: Smooth transitions, page changes
- **slower (500ms)**: Deliberate motion, complex animations
- **slowest (700ms)**: Dramatic effects, hero animations

### 3. **Animation Patterns**
- **fadeIn**: Content appearing, loading states
- **slideIn**: Elements entering from off-screen
- **scaleIn**: Interactive elements, buttons
- **bounce**: Notifications, success states

### 4. **Performance Considerations**
- Use `transform` and `opacity` for animations
- Avoid animating `width`, `height`, `margin`, `padding`
- Use `will-change` sparingly
- Respect `prefers-reduced-motion`

## Best Practices

### 1. **Purposeful Motion**
```tsx
// ✅ Good - Purposeful animation
<button 
  style={{ 
    transition: `transform ${tokens.motion.duration.fast} ${tokens.motion.easing.smooth}`,
    transform: isPressed ? 'scale(0.95)' : 'scale(1)'
  }}
  onMouseDown={() => setIsPressed(true)}
  onMouseUp={() => setIsPressed(false)}
>
  Interactive Button
</button>

// ❌ Bad - Unnecessary animation
<div style={{ 
  transition: 'all 0.3s ease',
  transform: isVisible ? 'rotate(360deg)' : 'rotate(0deg)'
}}>
  Confusing Animation
</div>
```

### 2. **Consistent Timing**
```tsx
// ✅ Good - Consistent timing
const cardStyle = {
  transition: `all ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth}`,
  transform: isHovered ? 'translateY(-4px)' : 'translateY(0)',
  boxShadow: isHovered ? tokens.shadows.hover : tokens.shadows.base
};

// ❌ Bad - Inconsistent timing
const cardStyle = {
  transition: 'transform 0.2s ease, box-shadow 0.3s ease-out',
  transform: isHovered ? 'translateY(-4px)' : 'translateY(0)',
  boxShadow: isHovered ? tokens.shadows.hover : tokens.shadows.base
};
```

### 3. **Accessibility Compliance**
```tsx
// ✅ Good - Respects motion preferences
const useReducedMotion = () => {
  const [prefersReducedMotion, setPrefersReducedMotion] = useState(false);
  
  useEffect(() => {
    const mediaQuery = window.matchMedia('(prefers-reduced-motion: reduce)');
    setPrefersReducedMotion(mediaQuery.matches);
    
    const handleChange = (e: MediaQueryListEvent) => {
      setPrefersReducedMotion(e.matches);
    };
    
    mediaQuery.addEventListener('change', handleChange);
    return () => mediaQuery.removeEventListener('change', handleChange);
  }, []);
  
  return prefersReducedMotion;
};

const AnimatedComponent = () => {
  const prefersReducedMotion = useReducedMotion();
  
  return (
    <div style={{
      transition: prefersReducedMotion ? 'none' : `transform ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth}`
    }}>
      Content
    </div>
  );
};
```

### 4. **Performance Optimization**
```tsx
// ✅ Good - Hardware-accelerated animation
<div style={{
  transform: isVisible ? 'translateX(0)' : 'translateX(-100%)',
  transition: `transform ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth}`,
  willChange: 'transform'
}}>
  Optimized Animation
</div>

// ❌ Bad - Layout-triggering animation
<div style={{
  marginLeft: isVisible ? '0' : '-100px',
  transition: `margin-left ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth}`
}}>
  Performance Heavy Animation
</div>
```

## How-To Guides

### Creating a Motion Utility System
```tsx
import { tokens } from '@/design-system/tokens';

const motion = {
  // Easing utilities
  ease: (duration: keyof typeof tokens.motion.duration, easing: keyof typeof tokens.motion.easing) => ({
    transition: `${duration} ${tokens.motion.easing[easing]}`
  }),
  
  // Duration utilities
  fast: (property: string) => ({
    transition: `${property} ${tokens.motion.duration.fast} ${tokens.motion.easing.smooth}`
  }),
  normal: (property: string) => ({
    transition: `${property} ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth}`
  }),
  slow: (property: string) => ({
    transition: `${property} ${tokens.motion.duration.slow} ${tokens.motion.easing.smooth}`
  }),
  
  // Animation patterns
  fadeIn: {
    opacity: 0,
    animation: `fadeIn ${tokens.motion.patterns.fadeIn.duration} ${tokens.motion.patterns.fadeIn.easing} forwards`
  },
  slideIn: {
    transform: 'translateX(-100%)',
    animation: `slideIn ${tokens.motion.patterns.slideIn.duration} ${tokens.motion.patterns.slideIn.easing} forwards`
  },
  scaleIn: {
    transform: 'scale(0.8)',
    animation: `scaleIn ${tokens.motion.patterns.scaleIn.duration} ${tokens.motion.patterns.scaleIn.easing} forwards`
  }
};

// CSS keyframes
const keyframes = `
  @keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
  }
  
  @keyframes slideIn {
    from { transform: translateX(-100%); }
    to { transform: translateX(0); }
  }
  
  @keyframes scaleIn {
    from { transform: scale(0.8); }
    to { transform: scale(1); }
  }
`;

// Usage example
const AnimatedCard = ({ children, isVisible }: { children: React.ReactNode; isVisible: boolean }) => (
  <div style={{
    ...motion.normal('all'),
    opacity: isVisible ? 1 : 0,
    transform: isVisible ? 'translateY(0)' : 'translateY(20px)'
  }}>
    {children}
  </div>
);
```

### Creating an Interactive Button Component
```tsx
interface AnimatedButtonProps {
  children: React.ReactNode;
  variant?: 'primary' | 'secondary';
  onClick?: () => void;
}

const AnimatedButton = ({ children, variant = 'primary', onClick }: AnimatedButtonProps) => {
  const [isHovered, setIsHovered] = useState(false);
  const [isPressed, setIsPressed] = useState(false);

  const getTransform = () => {
    if (isPressed) return 'scale(0.95)';
    if (isHovered) return 'scale(1.02)';
    return 'scale(1)';
  };

  const getStyles = () => {
    const baseStyles = {
      border: 'none',
      borderRadius: '6px',
      cursor: 'pointer',
      transform: getTransform(),
      transition: `all ${tokens.motion.duration.fast} ${tokens.motion.easing.smooth}`,
      willChange: 'transform'
    };

    const variantStyles = {
      primary: {
        backgroundColor: tokens.colors.primary[500],
        color: tokens.colors.white,
        boxShadow: tokens.shadows.sm
      },
      secondary: {
        backgroundColor: tokens.colors.white,
        color: tokens.colors.secondary[700],
        border: `1px solid ${tokens.colors.secondary[300]}`,
        boxShadow: tokens.shadows.sm
      }
    };

    return {
      ...baseStyles,
      ...variantStyles[variant]
    };
  };

  return (
    <button
      style={getStyles()}
      onClick={onClick}
      onMouseEnter={() => setIsHovered(true)}
      onMouseLeave={() => setIsHovered(false)}
      onMouseDown={() => setIsPressed(true)}
      onMouseUp={() => setIsPressed(false)}
    >
      {children}
    </button>
  );
};
```

## Common Patterns

### Fade In Animation
```tsx
const FadeIn = ({ children, delay = 0 }: { children: React.ReactNode; delay?: number }) => {
  const [isVisible, setIsVisible] = useState(false);

  useEffect(() => {
    const timer = setTimeout(() => setIsVisible(true), delay);
    return () => clearTimeout(timer);
  }, [delay]);

  return (
    <div style={{
      opacity: isVisible ? 1 : 0,
      transform: isVisible ? 'translateY(0)' : 'translateY(20px)',
      transition: `all ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth}`,
      willChange: 'opacity, transform'
    }}>
      {children}
    </div>
  );
};
```

### Staggered Animation
```tsx
const StaggeredList = ({ items }: { items: string[] }) => (
  <div>
    {items.map((item, index) => (
      <div
        key={index}
        style={{
          opacity: 0,
          transform: 'translateX(-20px)',
          animation: `slideIn ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth} forwards`,
          animationDelay: `${index * 100}ms`
        }}
      >
        {item}
      </div>
    ))}
  </div>
);
```

### Hover Card Animation
```tsx
const HoverCard = ({ title, content }: { title: string; content: string }) => {
  const [isHovered, setIsHovered] = useState(false);

  return (
    <div
      style={{
        padding: tokens.spacing[4],
        borderRadius: '8px',
        backgroundColor: 'white',
        border: `1px solid ${tokens.colors.secondary[200]}`,
        transform: isHovered ? 'translateY(-4px) scale(1.02)' : 'translateY(0) scale(1)',
        boxShadow: isHovered ? tokens.shadows.hover : tokens.shadows.base,
        transition: `all ${tokens.motion.duration.normal} ${tokens.motion.easing.smooth}`,
        cursor: 'pointer',
        willChange: 'transform, box-shadow'
      }}
      onMouseEnter={() => setIsHovered(true)}
      onMouseLeave={() => setIsHovered(false)}
    >
      <h3 style={{ marginBottom: tokens.spacing[3], margin: 0 }}>{title}</h3>
      <p style={{ margin: 0, color: tokens.colors.secondary[600] }}>{content}</p>
    </div>
  );
};
```

## Accessibility Guidelines

### 1. **Reduced Motion**
- Always respect `prefers-reduced-motion` media query
- Provide alternative visual feedback
- Test with motion reduction enabled

### 2. **Animation Duration**
- Keep animations under 500ms for most interactions
- Use longer durations only for dramatic effects
- Ensure animations don't interfere with user tasks

### 3. **Focus Management**
- Don't animate focus indicators
- Ensure animations don't interfere with focus
- Test with keyboard navigation

### 4. **Performance**
- Use hardware-accelerated properties
- Avoid layout-triggering animations
- Monitor animation performance

## Testing Checklist

- [ ] Animations respect `prefers-reduced-motion`
- [ ] All animations are purposeful and meaningful
- [ ] Performance is acceptable on all devices
- [ ] Animations don't interfere with accessibility
- [ ] Timing is consistent across similar interactions
- [ ] Animations enhance rather than distract from content
- [ ] Hardware acceleration is used appropriately
- [ ] Animation performance is monitored
