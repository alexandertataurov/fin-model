# Theming

FinVision uses CSS variables with Tailwind for themes.

## Themes

- Light (default)
- Dark

Switch themes using the toolbar.

## Design tokens

Semantic tokens are defined in `src/styles/globals.css` and exposed in Tailwind via `tailwind.config.js`.

Examples:

```css
:root {
  --background: #ffffff;
  --foreground: oklch(0.145 0 0);
  --primary: #030213;
  /* ... */
}

.dark {
  --background: oklch(0.145 0 0);
  --foreground: oklch(0.985 0 0);
  --primary: oklch(0.985 0 0);
  /* ... */
}
```

Use with Tailwind:

```tsx
<div className="bg-background text-foreground">
  <button className="bg-primary text-primary-foreground">Primary</button>
</div>
```
