# Charts Guidelines

## Purpose

Use charts to visualize trends and comparisons. Choose the chart type that best matches the data.

## Chart selection

- Bar: compare categories
- Line: trends over time
- Pie: composition (limited categories)
- Waterfall: sequential contributions

## Usage tips

- Keep axes and labels clear
- Limit colors; use semantic palette
- Provide empty/loading/error states
